# 第一章　起源与萌芽 · 先秦

> <span class="era">远古—公元前 221 年</span> 从经验的零散积累，到巫医分离与早期理论的孕育。

## 一、时代背景

在文字与系统理论出现之前，先民在采集、狩猎、农耕与日常生活中，已不断遭遇疾病与创伤，并在试错中积累起最朴素的医疗经验：哪些草木可以止痛，哪些动作可以接骨，火与热可以驱寒止痛。这些经验口耳相传，构成了中医最早的源头。

## 二、传说中的医药起源

中国关于医药起源的叙述，多归于上古圣王，反映了后世「圣人创制」的文化心理：

- **伏羲**：传说「画八卦」「制九针」，被附会为针灸之始。
- **神农**：传说「尝百草，一日而遇七十毒」，象征本草知识的经验来源。后世第一部药学专著即托名《神农本草经》。
- **黄帝**：传说与岐伯等臣子问答医理，中医理论奠基之作《黄帝内经》即托其名，「岐黄之术」遂成中医代称。

!!! note "如何看待这些传说"
    神农、黄帝并非真实的医学著作者，而是后人为知识寻找权威源头的**文化符号**。它们的价值不在于史实，而在于揭示：中医知识被认为来自对自然的长期观察与亲身试验。

## 三、从巫到医：医巫分离

上古医疗与巫术、祭祀长期交织，「巫医」一身二任。商代甲骨卜辞中已有大量关于疾病的记载——如「疾首」「疾目」「疾齿」「龋」（已认识到龋齿）——但多与占卜、祈禳并行。

随着经验医学的积累，理性的医疗手段逐渐从巫术中分离。春秋时期，名医**医和**「六气致病」之说、**医缓**「病入膏肓」之论，已基本摆脱鬼神解释，转向以自然因素解释疾病。这一**医巫分离**的过程，是中医走向独立学科的关键一步。

## 四、早期理论的孕育

先秦诸子的自然哲学，为中医理论提供了思想工具：

- **阴阳**学说：以对立统一解释自然与人体的运动变化。
- **五行**学说：以木火土金水的生克关系建立事物间的系统联系。
- **气**的观念：将「气」视为构成与维系生命的基本物质与动力。

这些观念被引入医学，逐渐形成藏象、经络、病机等理论雏形，为秦汉之际医学经典的成书做了准备。

## 五、出土文献的印证

20 世纪以来的考古发现，让我们得以窥见经典成书前的医学实态：

- **马王堆汉墓帛书**（成书或可早至先秦）：含《足臂十一脉灸经》《阴阳十一脉灸经》《五十二病方》等，反映经络学说尚未定型、以灸法与方药为主的早期面貌。
- **张家山汉简《脉书》、老官山汉墓医简**：进一步补充了早期经脉与诊法知识。

这些材料表明：先秦至秦汉之交，中医正处于**由经验向理论体系跃迁**的关键节点。

## 六、本章小结

先秦是中医的**萌芽期**：经验在积累，巫术在退场，哲学在注入。它没有留下完整的理论著作，却孕育了即将在秦汉迸发的整个理论框架。下一章，我们将看到《黄帝内经》如何把这些散点连成体系。
