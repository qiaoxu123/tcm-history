# 第八章　新生与现代化 · 当代

> <span class="era">1949 年至今</span> 制度重建、中西医结合、循证检验与走向世界，传统医学在科学时代寻找新位置。

## 一、时代背景

1949 年后，中医获得了制度性的保护与扶持，建立起完整的院校教育、医院与科研体系。与此同时，全球医学进入**循证医学**时代，「拿证据来」成为衡量任何疗法的通行标准。当代中医的核心命题，就是在保留自身特色的同时，**接受并参与现代科学检验**。

## 二、制度的重建

- **中医院校教育**：1956 年起，北京、上海、广州、成都等地相继成立中医学院，中医培养从师徒制转入现代高等教育。
- **中西医结合方针**：提出「团结中西医」「西医学习中医」，推动两套体系在临床与研究中协作。
- **中医医院与科研院所**：形成覆盖全国的医疗与研究网络。
- **《中华人民共和国中医药法》**（2017 年施行）：从法律层面规范中医药的行医、教育与管理。

## 三、青蒿素：传统线索 × 现代科学

最具标志性的成就，是**屠呦呦**团队从抗疟研究中发现**青蒿素**：

1. 团队系统筛查古代方书，从葛洪《肘后备急方》「青蒿一握……绞取汁」一句获得关键启发；
2. 据「绞汁冷服」的细节，推断有效成分**不耐高温**，遂改用**低温乙醚提取**，活性大增；
3. 经动物实验、临床试验验证疗效，最终分离出青蒿素这一全新结构的抗疟药。

2015 年，屠呦呦因此获**诺贝尔生理学或医学奖**，成为首位获科学类诺奖的中国本土科学家。

!!! note "第一性原理的最佳范本"
    青蒿素既不是「中医的胜利」也不是「对中医的否定」，而是**传统经验提供假说、现代科学完成验证**的协作典范：
    
    - 古籍提供了**线索**（青蒿、冷服），但没有提供有效成分、剂量、机理；
    - 现代化学、药理学、临床试验提供了**证据与产品**；
    - 关键转折（低温提取）恰恰来自对经验细节的**科学化解读**。
    
    它给中医现代化指明了一条务实路径：把传统当作**假说的富矿**，用现代方法去伪存真、提纯升级。

## 四、现代研究：成就与争议并存

当代对中医的科学研究呈现复杂图景，本书主张如实呈现两面：

- **已获证据支持的方向**：部分单味药/成分（青蒿素、砒霜治疗急性早幼粒细胞白血病、黄连素等）经严格研究证实有效；针刺在某些疼痛、恶心等适应症上有一定循证证据。
- **证据不足或阴性的方向**：大量复方与疗法缺乏高质量随机对照试验；部分已发表研究存在样本小、设计缺陷、发表偏倚等问题；某些经典理论（经络的物质实体等）至今未获现代解剖生理学证实。
- **安全性问题**：马兜铃酸肾毒性、含重金属成药（朱砂、雄黄）、中药肝损伤等，是必须正视的现实风险。

!!! warning "批判性视角（全书立场的落点）"
    对当代中医，最不可取的两种态度是**全盘神化**与**全盘否定**。遵循第一性原理，应当：
    
    1. **分而论之**：不把「中医」当作铁板一块去整体肯定或否定，而是逐方、逐法、逐适应症地用证据评判；
    2. **机理与疗效分开**：一个疗法即使有效，其古代理论解释未必正确；理论动听也不能替代疗效证据；
    3. **安全优先**：疗效未明而毒性已知者，应审慎乃至弃用；
    4. **拥抱检验**：真正有价值的传统知识，不怕被研究——青蒿素证明了这一点。

## 五、走向世界

- 针灸已被世界卫生组织关注，并在多国作为补充疗法使用；
- 2019 年，世界卫生组织《国际疾病分类》第十一版（ICD-11）首次纳入传统医学章节（此举亦伴随国际医学界关于「证据」的持续争论）；
- 中药的国际化面临质量标准、成分明晰、循证证据等多重门槛。

## 六、结语：历史给今天的启示

回望两千余年，中医的生命力始终来自**实践中的不断试错与自我修正**——从医巫分离，到金元争鸣，到温病突围，到青蒿素。它最宝贵的遗产，或许不是某条具体的结论，而是一种**面对真实病情、勇于质疑成说、以效验为归依**的精神。

让这一传统在科学时代继续有所贡献的唯一道路，不是固守经典，也不是全盘抛弃，而是：**以第一性原理审视它，用现代证据淬炼它，让真正有效的部分以可被世界理解的方式流传下去。**
