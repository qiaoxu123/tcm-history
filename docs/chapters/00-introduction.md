# 导论

## 一、什么是中医

「中医」一词，狭义指以《黄帝内经》理论为核心、以辨证论治为方法、以中药与针灸等为手段的中国传统医学；广义则涵盖中国各民族传统医学（如藏医、蒙医、维医）共同构成的医学传统。本书所述以汉族传统医学为主线，兼及与之交融的其他医学传统。

中医学有几个区别于近代西方医学的显著特质：

- **整体观念**：视人体为一个有机整体，并强调人与自然、社会环境的统一（「天人相应」）。
- **辨证论治**：以「证」而非单一「病」为治疗对象，因人、因时、因地制宜。
- **功能视角**：藏象、经络、气血等概念更着眼于功能联系而非纯解剖结构。
- **实践积累**：理论与临床长期互证，本草与方剂在反复实践中筛选积累。

## 二、为什么要读中医发展史

理解一门学问，最好的方式之一是回到它形成的历史现场。读中医史，至少有三重意义：

1. **看清理论的来路**：阴阳五行、藏象经络并非凭空而来，而是先民观察自然与人体、并借助当时哲学语言概括的产物。
2. **理解学派的分歧**：寒凉、攻下、补土、滋阴……金元以降的学派争鸣，本质是不同医家对同一难题的不同解答，了解其背景方能不偏执一端。
3. **审视当下的处境**：近代以来中西医的碰撞、存废之争与现代化探索，至今仍深刻影响着中医的命运与形态。

## 三、分期方法

历史分期没有唯一标准。本书综合学界通行做法，按**朝代更迭与学术形态**将中医发展划分为八个阶段：

| 阶段 | 时期 | 学术形态关键词 |
|------|------|----------------|
| 起源与萌芽 | 远古—先秦 | 经验积累、巫医分离 |
| 理论奠基 | 秦汉 | 经典成书、体系确立 |
| 充实发展 | 魏晋南北朝 | 专科细化、文献整理 |
| 集成鼎盛 | 隋唐 | 官修典籍、医学教育 |
| 学派争鸣 | 宋金元 | 校勘普及、四大家 |
| 总结普及 | 明清 | 集大成、温病学派 |
| 碰撞与变革 | 近代 | 西学东渐、存废之争 |
| 新生与现代化 | 当代 | 中西结合、科学化 |

需要强调的是，分期是为叙述方便而作的**人为切割**，历史本身是连续的：理论奠基期的思想会延续千年，明清的总结也植根于此前的层层积累。读者宜在心中保留这条连续的长河。

## 四、史料与考据说明

本书所依据的史料，主要包括：

- **传世医学典籍**：如《黄帝内经》《伤寒杂病论》《本草纲目》等，是第一手的学术文献。
- **正史与方志**：如《史记·扁鹊仓公列传》、历代《艺文志》《经籍志》中的医书著录。
- **出土文献**：如马王堆汉墓帛书、张家山汉简、老官山汉墓医简等，可订正传世记载。
- **现代研究**：近现代中医史学者的考证成果。

涉及传说人物（如神农、黄帝）与托名著作时，本书会注明其**传说性质**与**成书年代的学术争议**，以史实为准，不轻信附会。

## 五、写作立场：第一性原理，而非经典崇拜

本书是一部**历史**，不是一部教科书，更不是辩护词。我们尊重经典作为历史文献的价值，但**不把「经典如是说」当作「事实如是」**。贯穿全书，我们坚持以下几条第一性原理：

1. **区分三类陈述**。叙述时严格分开：
    - *历史事实*（某书于某代成书、某医家确有其人其事）——以史料与考古为据；
    - *历史观点*（古人对人体与疾病的理论解释）——如实呈现，但标明这是「当时的认识」；
    - *疗效与机理断言*（某法某药「能治某病」「其理为某」）——区分「有长期实践经验」与「经现代研究证实」，二者不可混同。

2. **追问证据，而非诉诸权威**。某结论被沿用千年，只说明它有历史影响力，不等于它正确。「医圣说」「经典载」不是论证的终点，而是需要进一步考据与检验的起点。

3. **回到产生知识的现场**。阴阳五行、藏象经络等概念，应放回其**历史语境**理解：它们是古人借助当时哲学语言对观察到的现象作出的概括与建模，有其解释力，也有其时代局限。既不神化，也不简单斥为迷信。

4. **正视争议与失败**。学派分歧、被淘汰的疗法、近代的存废之争、现代研究的阴性结果，都是历史的一部分，本书不回避、不粉饰。

5. **承认未知**。对机理不明、证据不足或学界无定论之处，明确说「尚不清楚」「存在争议」，而非用经典话语填补空白。

!!! warning "一句话立场"
    读经典是为了**理解中医如何思考**，而不是为了**接受中医的全部结论**。本书希望读者读完后，既能体会这一传统的智慧与韧性，也能以现代理性的眼光审视它的主张。
