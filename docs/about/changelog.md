# 更新日志

本页镜像仓库根目录的 `CHANGELOG.md`，记录版本级变更。

## [0.1.0] - 2026-06-20

### Features
- 确立全书脉络：编年史八章 + 专题卷五篇 + 附录四篇的整体结构。
- 完成编年史八章（先秦至当代）的初稿正文。
- 完成专题卷与附录的结构化骨架（含待扩充清单）。
- 搭建 MkDocs Material 文档站，支持网页浏览与 PDF 导出双形态。
- 确立全书写作立场：**第一性原理，不一味依据经典**（见导论 §五，并贯穿各章「批判性视角」框）。

### Design Rationale
- 选用 MkDocs Material：Python 生态、中文友好、维护成本低、PDF 插件成熟，契合「开源文档式书籍」定位。
- 采用「编年为主、专题为辅」体例：兼顾时间主线与主题纵深，是中医史学界通行做法。
- 内容许可 CC BY-SA 4.0、代码许可 MIT：便于开源协作与二次传播。

### Notes & Caveats
- 各章为**初稿**，史实、年代、引文待逐条核对并补脚注出处。
- 参考文献为占位条目，正式引用前需核实版本页码。
- PDF 导出依赖 weasyprint 的系统库（pango 等），见贡献指南。
