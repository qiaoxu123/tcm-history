# 中医发展历史

> 一部开源、可持续修订的中医学发展通史，提供**网页**与 **PDF** 两种阅读形态。

中医学是中华民族在数千年与疾病斗争中积累、提炼并体系化的医学知识体系。它既是一门临床实践之术，也是一套关于生命、自然与人体的整体哲学。本书试图沿着历史的脉络，呈现中医从远古萌芽到当代现代化的完整演进，回答三个问题：

- **它从哪里来？** —— 理论与方药体系如何在两千余年里逐步奠基、充实、争鸣、总结。
- **它如何成形？** —— 历代关键医家、典籍与制度在其中扮演了怎样的角色。
- **它走向何方？** —— 在与西方医学的碰撞中，中医如何存续、变革并走向世界。

## 全书结构

本书采用**编年为主、专题为辅**的体例：

- **编年史（八章）**：按先秦、秦汉、魏晋南北朝、隋唐、宋金元、明清、近代、当代八个时期，勾勒主线。
- **专题卷（五篇）**：就本草、针灸、方剂、典籍、中外交流五条线索作纵向梳理。
- **附录**：历代医家年表、重要典籍一览、术语表与参考文献，便于检索查考。
