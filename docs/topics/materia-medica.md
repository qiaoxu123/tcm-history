# 专题一　本草学发展史

> 纵向梳理中药知识从经验到典籍、再到现代药理研究的演进。本篇与编年各章互为经纬。

## 主线脉络

| 时期 | 标志性著作/事件 | 关键进展 |
|------|----------------|----------|
| 汉 | 《神农本草经》 | 365 药，立四气五味、三品、七情 |
| 南北朝 | 陶弘景《本草经集注》 | 730 药，首创自然属性分类 |
| 唐 | 《新修本草》 | 850 药，首部国家药典，图文并茂 |
| 宋 | 《证类本草》 | 集前代之大成，保存大量佚文 |
| 明 | 李时珍《本草纲目》 | 1892 药，纲目分类，博物学巅峰 |
| 清 | 赵学敏《本草纲目拾遗》 | 补《纲目》之缺，收新增药与民间药 |
| 近现代 | 化学成分与药理研究 | 提取有效成分、阐明机理（如青蒿素、麻黄碱） |

## 第一性原理视角（本篇贯穿）

- **药效来自成分，而非「药性」标签**：四气五味是古人的经验分类与说理框架，现代应追问其化学成分与药理机制。
- **经验须经毒理重审**：朱砂（汞）、雄黄（砷）、马兜铃（马兜铃酸）等传统药物的安全性，必须以现代毒理学为准。
- **「久服轻身」类表述**多属时代局限，不可作为用药依据。
