# 专题三　方剂学发展史

> 从单方、验方到经方、时方，再到成药标准化的演进。

## 主线脉络

| 时期 | 标志 | 关键进展 |
|------|------|----------|
| 先秦—汉 | 《五十二病方》《伤寒杂病论》 | 由经验方到「经方」与配伍法度（君臣佐使） |
| 唐 | 《千金方》《外台秘要》 | 大型方书集成，方剂数量激增 |
| 宋 | 《太平惠民和剂局方》 | 官修成药标准，方剂走向成药化、标准化 |
| 金元 | 各家自创新方 | 学派主张催生新方（如补中益气汤、丹溪诸方） |
| 明清 | 温病诸方、《医方集解》 | 温病学派新方系统化，方论体系成熟 |
| 现代 | 中成药、配方颗粒 | 工业化生产、质量标准与循证评价 |

## 第一性原理视角

- **复方的复杂性**：一首复方含多种成分，疗效评价比单味药更难；「经验有效」需以现代临床试验区分真实效应与安慰剂效应、自愈与回归均值。
- **配伍理论 vs 药物相互作用**：「君臣佐使」「七情」是古代配伍经验的概括，现代应以药代动力学、药物相互作用研究检验其中哪些成立。
- **剂量与质量**：古今度量不同、药材产地与炮制差异巨大，照搬古方剂量需谨慎。
