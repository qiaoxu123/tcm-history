# 专题二　针灸学发展史

> 从砭石、灸法到经络腧穴体系的建立、标准化与现代研究。

## 主线脉络

| 时期 | 标志 | 关键进展 |
|------|------|----------|
| 远古 | 砭石、灸法 | 以石、火刺激体表治病的原始疗法 |
| 先秦—汉 | 马王堆脉灸经、《灵枢》 | 经脉学说由不定型走向系统（《灵枢》又称《针经》） |
| 晋 | 皇甫谧《针灸甲乙经》 | 现存最早针灸专著，定 349 穴 |
| 宋 | 王惟一针灸铜人、《铜人腧穴针灸图经》 | 腧穴定位标准化，用于教学考试 |
| 明 | 杨继洲《针灸大成》 | 集大成，流传最广 |
| 现代 | 针刺镇痛、WHO 标准经穴定位 | 进入实验研究与国际标准化 |

## 第一性原理视角

- **经络的「实体」问题**：经络在现代解剖学中未找到对应的独立结构；将其理解为古人对体表—内脏功能联系的**经验性建模**更为稳妥。
- **针刺疗效需分适应症**：现有循证证据对针刺在某些疼痛、术后恶心呕吐等方面较为支持，对许多其他适应症证据不足；「安慰针/假针对照」研究是关键方法。
- **机理研究**：内啡肽释放、神经调节等是当前较受关注的解释方向，但尚不能解释全部声称疗效。
