# 专题五　中外医学交流史

> 中医并非孤立发展——它在与周边及世界医学的双向交流中吸收、外传与被检验。

## 主线脉络

| 时期 | 方向 | 内容 |
|------|------|------|
| 汉唐 | 引进 | 丝路输入域外香药（乳香、没药等）、印度/波斯医学 |
| 唐 | 外传 | 鉴真东渡传医日本；典籍传入朝鲜、日本 |
| 宋元 | 双向 | 阿拉伯医药（回回药方）输入；海上贸易扩大药材交流 |
| 明清 | 外传 | 人痘接种术经丝路传至奥斯曼、欧洲，启发牛痘 |
| 近代 | 引进为主 | 西医东渐，中西医汇通与冲突（见近代章） |
| 当代 | 外传 | 针灸、中药走向世界；ICD-11 纳入传统医学章节 |

## 第一性原理视角

- **交流是双向且功利的**：传播与采纳往往取决于**实际效用**而非理论优劣（如人痘外传、阿拉伯香药输入）。
- **比附的陷阱**：跨体系比较（如中医脏腑 vs 西医器官）易生牵强附会，应以概念各自的语境为准。
- **国际化的真门槛**：中药/针灸走向世界，最终仍要跨过**质量标准与循证证据**这道关。
