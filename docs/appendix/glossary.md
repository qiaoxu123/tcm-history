# 附录三　术语表

> 中医常见术语的**历史语境**释义。本表力求中立：解释古人「用这个词指什么」，并在必要处标注其与现代科学概念的关系，不预设其客观成立。

## 基础理论

- **阴阳**：古代用以概括事物对立统一两方面的哲学范畴，引入医学后用于说明人体生理、病理与诊治。属**说理框架**。
- **五行**：木火土金水及其生克关系，用以建立脏腑、季节、情志等之间的系统联系。属**类比模型**。
- **气**：古人理解的构成与维系生命活动的基本物质与动力，含义随语境而变（元气、宗气、营气、卫气等）。
- **精 / 血 / 津液**：人体基本物质，与「气」共同构成生命活动的物质基础（古代框架）。

## 藏象与经络

- **藏象（脏象）**：通过外在征象推测内在脏腑功能的学说。其「脏腑」是**功能单位**，与现代解剖器官不完全对应。
- **经络**：古人设想的气血运行通路（十二经脉、奇经八脉等）。现代解剖学未发现其独立实体结构，宜理解为经验性功能建模。
- **腧穴（穴位）**：体表特定刺激点，针灸施术部位。

## 病因病机

- **六淫**：风、寒、暑、湿、燥、火六种外感致病因素（古代外因分类）。
- **七情**：喜、怒、忧、思、悲、恐、惊七种情志，过极可致病（古代内因分类）。
- **戾气（疠气）**：吴又可提出的、有别于六淫的「物质性」致病因子，被视为对传染病病原的早期推想。

## 诊治

- **四诊**：望、闻、问、切四种诊察方法。
- **辨证论治**：以「证」（一组症状体征的概括）为对象、确定治法的方法论，是中医临床的核心特征。
- **君臣佐使**：方剂配伍中药物主次与协同关系的组织原则。

!!! warning "使用说明"
    本表的释义意在**帮助理解历史文本**，而非主张这些概念在现代科学意义上成立。凡涉及与现代医学的对应，均按第一性原理审慎标注「尚无证据 / 部分对应 / 存争议」。
